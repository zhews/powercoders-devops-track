## What is Jaeger?

- Distributed Tracing Platform
- Open Source
- Maps Flows and Data

---

## Baggage

- HTTP Header
- Contains Unique Identifier
- Every Service adds Tracing Information

---

## Trace

- Data or Execution Path through System
- Directed Acyclic Graph of Spans

---

## Spans

- Logical Unit of Work
- Contain
  - Operation Name
  - Start Time
  - Duration
